Sobre

En este repositorio verás una lista de mis proyectos públicos separados por carpetas, el cual será mi portafolio donde podrás ver proyectos web públicos hechos en HTML5, CSS3, JavaScript y PHP7 donde voy a incluir proyectos realizados en Laravel 5, Symfony 4, WordPress, Bootstrap 3.3 y 4.1, NodeJS, TypeScript, Angular y jQuery. Lo que más pueda aprender lo subiré aquí mismo para compartirlo y que otros puedan aprender con mi experiencia como desarrollador web; en los proyectos también aplicaré cosas de RUP, también Scrum para pulirlos bien, Kanban Board en el mismo GitHub y dejaré documentación interesante adicional como un plus.

PHP7: Laravel 5.x, Symfony 4, WordPress.

JavaScript: TypeScript, jQuery, Angular 7, NodeJS, ExpressJS.

HTML5 & CSS3: Bootstrap.
